🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  重要：VXE Table 是可选组件，需要在应用中单独集成

1. 安装依赖（在需要使用的应用中）:
   cd web/your-app
   pnpm add vxe-table vxe-table-plugin-antd xe-utils

2. 初始化配置 (main.js):
   import { initVxeTable } from '@k8s-agent/shared/config/vxeTable'
   initVxeTable()

3. 创建组件（在应用中，不是 shared）:
   参考 VXE_TABLE_INTEGRATION.md 创建完整的 VxeBasicTable 组件

4. 使用 Hook:
   import { useVxeTable } from '@k8s-agent/shared/hooks'
   const [gridRef, gridApi] = useVxeTable(options)
